# Elektron Industrial

Gas-to-EV conversion services — Victorville, CA.

## Developer environment (EDE)

Clone → configure workstation:

```bash
./scripts/setup.sh
# or
npm run setup
npm run doctor
npm run dashboard
```

See [`ENVIRONMENT.md`](ENVIRONMENT.md), [`docs/README.md`](docs/README.md), and [`PROJECT_MANIFEST.json`](PROJECT_MANIFEST.json).  
EDE is additive infrastructure — it does **not** move application folders.
