## Architectural Safety Amendments & Engineering Rules

The following structural corrections and explicit safety rules are incorporated into the Build Engine to govern the boundary between high-voltage distribution validation and traction system enablement.

### 1. High-Voltage Isolation Testing Infrastructure Rules

To eliminate lethal flash hazards and unmitigated short-circuit conditions during insulation validation:
Isolation Fault Test Mandate: Isolation fault injection may only be performed using an approved, rated, current-limited HV isolation-test fixture or an IMD supplier-supported test method. Manual, ad-hoc resistance insertion or raw wire jumpers onto live high-voltage rails is strictly forbidden.

### 2. Manual E-Stop Invalidation & Retry Logic

To prevent dangerous control-loop recovery behavior following a physical emergency intervention:
E-Stop Lockout Rule: Automatic retry or autonomous re-energization following a manual E-stop event is strictly forbidden across all software layers, state machines, and hardware nodes. Any occurrence of an automatic retry sequence constitutes an immediate, catastrophic gate failure.

### 3. Traction Inverter Enable Readiness Constraints (Gate 05M-A)

To maintain complete hardware safety across varying vendor firmware implementations, Gate 05M-A enforces a non-negotiable state envelope:
- No traction torque command may be generated by the VCU or processed by the inverter.
- No intentional motor rotation or physical vehicle chassis movement is permitted.
- No driver torque authority (cabin pedal tracking to inverter command) is enabled.
- No power-stage bridge switching or PWM generation is permitted unless supplier documentation explicitly defines it as part of a safe, zero-torque readiness state and engineering formally approves it. The state machine validates the supplier-defined inverter ready / torque-disabled state strictly under these boundaries.

## Gate 05M-B — No-Load Motor Spin Validation

This gate governs the first physical rotation of the traction motor shaft inside the vehicle chassis. At this stage, the motor shaft is completely uncoupled from the vehicle's secondary drivetrain elements (such as driveshafts, axles, or gearboxes) to eliminate external mass inertia and load dynamics.

This phase focuses exclusively on calibrating the physical resolver offset angle, verifying the electrical phase rotation sequence, monitoring phase current balance, and testing over-current trip responses under low-torque spin profiles.

```
                    [GATE 05M-B NO-LOAD SPIN BOUNDARY]
          (Motor Shaft Uncoupled - LIVE HV - CONTROLLED LOW TORQUE)
                                   │
       ┌───────────────────────────┼───────────────────────────┐
       ▼                           ▼                           ▼
[Resolver Alignment]        [Phase Sync & Sine Audit]   [Dynamic Safety Trips]
- Inject low-current spin   - Verify balanced phase A,B,C- Validate over-speed trip
- Calibrate angular offset  - Monitor current distortion - Watchdog/E-stop drop checks
```

### 1. Resolver Angular Offset Calibration

- Alignment Tuning: To ensure efficient torque vector generation and prevent current runaway conditions, the electrical offset angle between the motor rotor magnetic field and the resolver stator poles must be calibrated.
- Verification Procedure: The inverter injects a low-amplitude current vector to index the rotor position. The physical offset value returned by the calibration routine must be written to non-volatile memory and validated across multiple directions of rotation.

### 2. Electrical Phase Sync & Harmonic Audit

- Phase Rotation Sequence: The physical cabling of phases U, V, W from the inverter to the motor must perfectly match the internal software coordinate frame transformation. A positive torque request must result in a positive physical rotation vector.
- Current Balancing: During low-speed spinning, the active phase current waveforms must be monitored using high-frequency current probes. The current waveforms must form a balanced three-phase system with minimal total harmonic distortion (THD).
