Gate 05I-C — Low-Voltage Communications Integration & Network Robustness
This gate establishes the complete verification protocol for the low-voltage communication buses (CAN_2, CAN_3, and diagnostic/display lines) in an isolated, non-powered-HV bench environment.
Before any controller is allowed to connect to the vehicle's physical networks, it must prove that its communication stacks, frame-handling engines, transceiver configurations, and sleep/wake controllers are robust against electrical and logical network faults.

3. Verification Protocol Specifications & Stress Profiles
A. CAN Bus Physical Loading Parameters
Physical Line Termination: Both CAN_2 and CAN_3 must be physically terminated on the bench using two 120 ohm metal-film resistors (1% tolerance) placed at the extreme physical ends of the wiring loop.
Simulated Network Length: The physical wire run of the bench harness must use twisted-shielded pair (110 turns/meter minimum) and simulate at least 10 meters of harness length to introduce realistic propagation delays.
Bus-Stress Distribution: [Nominal Traffic] 0-30% bus load (Nominal system operation); [Margin Traffic] 30-75% bus load (Simulated network stress); [Stress Level] 75-90% bus load (Worst-case stress limit - NO safety-critical message drops allowed)
B. DBC Version Matching & Scaling Rules
All target controllers must use a unified, version-controlled DBC database file (PROJECT_NAME_CAN_vX.Y.Z).
Bit-Packing Integrity: Verify Big-Endian (Motorola) vs. Little-Endian (Intel) formatting alignments. The testing tool must inject alternating bit-patterns (0x5555 / 0xAAAA) into all control payloads to confirm zero byte-swapping or bit-shifting errors at the parser boundary.
Scale Factor Safeguards: Physical Value = (Raw Integer x Factor) + Offset. Every critical input received by the VCU must run through saturation limiters. Any decoded value outside the physical hardware capabilities must instantly trip a network-validation DTC and command a zero-torque safe state.
C. Sleep/Wake Transition Limits
[12V Current (mA)]: 150mA at t_0 (Ignition Off), Transition Window (Target: <= 2.0s), Sleep Target (<= 1.0mA).
