# Research claims (Hard Rule 15)

Every **major** claim gets its own record:

| Field | Required |
|---|---|
| Claim ID | yes |
| Claim | yes |
| Evidence Level | yes |
| Configuration | yes |
| Supporting Sources | yes |
| Conflicting Sources | yes (may be empty) |
| Status | yes |
| Next Verification Step | yes |

Index: `CLAIM_REGISTER.json` · Schema: `schemas/research-claim.schema.json`
