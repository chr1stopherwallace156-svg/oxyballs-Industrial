# Component Sourcing Passports (CSP)

Modular component-level maturity trackers under deconstructed sourcing (Hard Rule 17).

- IDs: `CSP-######` (immutable). Aliases like `AID-COMP-01` are legacy operator labels.
- Operator saturation % bars are **research draft estimates**, not verified engineering confidence (Hard Rule 5).
- Related AIDs/AJRs/KGs link assets, purchase gates, and gaps without collapsing the vehicle into one unit.
