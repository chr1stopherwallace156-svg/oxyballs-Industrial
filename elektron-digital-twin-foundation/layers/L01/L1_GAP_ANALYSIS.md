## Critical Gaps

### G1 — Scan Reference Material (Dana S110)

- Need: high-resolution 3D scan or dense photo set of Dana S110 rear axle housing
- Why: irregular cast-iron shape cannot be invented from brochure silhouettes
- Blocks: axle mesh fidelity, micro-scan inset completion

### G2 — Placard Decal Information

- Need: clear photographic references for tire-pressure warnings and payload rating labels in driver door jamb
- Why: exact tire size / pressures / ratings are configuration-binding
- Blocks: wheel/tire dossier verification; intake placard fields

### G3 — Under-Cab Structural Rib Data

- Need: high-quality photos of cab underside where cab floor meets frame rails
- Why: visible from Ring 1 low angles; easy to get wrong
- Blocks: low-angle photogrammetry completeness; cab-to-frame interface modeling later

### G4 — OEM Layout Book Archive

- Need: archived 2019 Ford Truck Body Builder Layout Book pages for 145.3 WB / 60 CA chassis cab
- Why: frame width, CA, connector locations, dual track must be Tier 1
- Blocks: dimensional promotion beyond DRAFT

### G5 — Front Axle Architecture Tag

- Need: authoritative confirmation monobeam vs twin-I-beam for this 4x2 chassis cab
- Why: object hierarchy and landmarks depend on it
- Blocks: front axle object ID finalization

### G6 — Transit Bumper Policy

- Need: owner decision on temporary rear transit hardware inclusion
- Why: changes rear capture targets and object set
- Blocks: rear object freeze
